# ZeroDay Item Duplicator for Donut SMP





# 🚀 Advanced Item Duplication Tool for Donut SMP
ZeroDay is a powerful item duplication tool designed specifically for Donut SMP players who want to gain an advantage by duplicating items in their chests.

# ✨ Key Features
Chest Selection: Select any chest and duplicate its contents
Item Duplication: Duplicate any item in the game without limits
Stealth Mode: Operates silently without detection
Auto-Collection: Automatically collects duplicated items
Multi-Chest Support: Duplicate across multiple chests simultaneously
Quantity Control: Specify exact quantities to duplicate
Bypass Protection: Advanced algorithms to bypass server anti-cheat
Real-time Preview: See items before duplication
Backup System: Creates backups of original chest contents

# 🎮 Donut SMP Compatibility
ZeroDay is specifically optimized for Donut SMP servers with features including:

Custom chest interaction protocols
Server packet manipulation
Anti-detection mechanisms
Lag-free duplication process
Compatibility with all chest types

# 📥 Installation
Install python for Windows

Install Git for Windows

Clone the repository:

git clone https://github.com/ZeroDay-Dev/ZeroDay-Dupe.git

cd ZeroDay-Dupe

Install dependencies:

pip install -r requirements.txt

Run the application:

python main.py

# 🛠️ How to Use
Launch ZeroDay and connect to your Minecraft client
Stand near the chest you want to duplicate items from
Select the chest in the ZeroDay interface
Choose items and quantities to duplicate
Click "Duplicate" and watch the magic happen
Collect your duplicated items from the chest
